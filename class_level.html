Detailed Description

This is a level in raw data. This class is ment to encapsulate all the information required for a level and makes it avaible to the LevelScene. This class holds the Sceney aka the background and forground items as well as loads key information for the level from the .tmx file such as Enemies Gold and the CrumbleBlock.
