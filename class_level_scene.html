Detailed Description

This is the inheriting sub class of Scene and as such overloads the appropratie methods (e.g. handleEvent, render). This class is responcable for updating and rendering everything in the scene while also handlingEvents from the keyboard or input from the controller. The LevelScene has a Level object and is responcible for loading it into memory. The Level scene controls the fixed time step and handling the box2d world.
