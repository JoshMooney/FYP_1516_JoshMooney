Detailed Description

This is an instance of the XBOX controller which uses xinput v9.1.0 to detect and store information on the first controller connected to the system. This class will be passed around by the game to different scenes to detect input only if the controller is actually connected. Otherwise the keyboard is passed around for input checking.
